## Zed Settings

Para usar este `settings.json` no Zed, copie e cole o conteúdo no arquivo de configuração do usuário: Linux em `~/.config/zed/settings.json`, macOS em `~/Library/Application Support/Zed/settings.json` e Windows em `%APPDATA%\Zed\settings.json`; se o arquivo não existir, crie, salve e reabra o Zed para aplicar as mudanças, e caso já tenha configurações próprias, faça a mesclagem dos campos.
